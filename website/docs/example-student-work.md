# Example Student Work

Use [Example Artifacts](./example-artifacts.md) for realistic examples of:

- Systems Log entries
- terrarium observations
- lifecycle audits
- open-loop diagnoses
- Specs Sheets
- failure-mode notes
- pitch outlines

These examples are meant to feel achievable for ages 8–12. They are not answer keys and should be shown selectively.
