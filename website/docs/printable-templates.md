# Printable Templates

Use [Templates and Worksheets](./templates-and-worksheets.md) for copyable or printable versions of:

- Systems Log pages
- Environmental Checkpoint sheets
- quick environmental checks
- environmental data checks
- terrarium logs
- reservoir and flow planners
- away audits
- resource pool trackers
- capstone proposal pages
- Make-the-Plan-Real sheets
- failure-mode and stakeholder worksheets
- learner self-checks
- attribution and AI-use notes
- honest environmental systems project checklists
- pitch outlines and version 2.0 reflections

The templates are intentionally plain so they can be printed, copied into notebooks, or read aloud while a student fills them in.
